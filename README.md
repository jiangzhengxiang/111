微信小程序，通过静态网页移植到uinapp 
 1. 🤖 智能 AI 引导者（LLM 模块）
   * 深度集成 DeepSeek：支持 deepseek-chat 与 deepseek-reasoner 双模型切换。
   * 流式传输 (Streaming)：文字逐字实时呈现，消除等待焦虑，体验流畅自然。
   * 本地知识库增强 (RAG)：用户可选择“情绪”、“欲望”、“放松”三个核心模块。勾选后，AI
     会自动“阅读”相关深度理论，基于专业背景知识为用户提供针对性的建议。
   * 可视化思考态：内置 CSS3 环形加载动画，清晰展示 AI 的思考过程。

  2. 📚 智慧知识库 (Knowledge Base)
   * 全量理论支持：完整录入圣多纳释放法、东方智慧（庄子、老子）、现代正念研究（杨定一、莫子）的原始文本。
   * 沉浸式排版：采用模块化段落、线性引言块设计，支持 white-space:
     pre-wrap，完美保留了理论的排版层次，适合在手机端深度阅读。

  3. 🌬️ 科学呼吸引导 (Breathing Guide)
   * 精准自定义：支持吸气、闭气、呼气三个阶段的时长调节。
       * 吸气/呼气：0s - 12s 自由调节。
       * 闭气：0s - 120s 极限拉伸。
   * 动态视觉球：通过高性能 WXSS
     动画，引导球随呼吸节奏缩放。在“闭气”阶段，加入高频微震动（Vibrate）特效，模拟生理层面的能量蓄积感。

  4. 🍃 释放流引导 (Release Workflow)
   * 交互式决策树：基于一套严密的逻辑算法，通过“是/否/继续”的简单交互，引导用户抽离头脑故事，回归身体感受。
   * 进度感知：内置实时进度条与操作回溯（撤销）功能，记录内在探索的每一步。
